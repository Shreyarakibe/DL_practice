

#  Deep Learning Practice Repository

A collection of hands-on deep learning experiments implemented using **Python**, **TensorFlow/Keras**, and **Jupyter Notebooks**.
This repository includes foundational to advanced deep learning models such as **FNN, CNN, Autoencoders, Word Embeddings, and Transfer Learning**.

---

##  Contents

### **1 FNN.ipynb — Feedforward Neural Network**

* Basics of neural networks
* Dense layers & activation functions
* Backpropagation
* Regularization
* Model training & evaluation

---

### **2. DL1.ipynb — Deep Learning Basics / Classification**

* Dataset preprocessing
* Building deep neural networks
* Loss functions & optimizers
* Training visualization

---

### **3 Image_classification_CNN.ipynb — Convolutional Neural Networks**

* Convolution, pooling, flattening
* Image augmentation
* CNN model building
* Overfitting control using dropout
* Accuracy & loss plotting

---

### **4 Transfer_learning_VGG16.ipynb — Transfer Learning with VGG16**

* Using VGG16 as a pre-trained model
* Feature extraction
* Fine-tuning
* Comparing accuracy with & without transfer learning

---

### **5 Autoencoder.ipynb — Autoencoder for Dimensionality Reduction**

* Encoder–decoder architecture
* Latent space representation
* Image reconstruction
* Denoising autoencoder

---

### **6 CBOW.ipynb — Word Embeddings (NLP)**

* Continuous Bag-of-Words implementation
* Creating word vectors
* Context window processing
* Semantic similarity

---

##  Technologies Used

* Python 3.x
* TensorFlow / Keras
* NumPy
* Matplotlib
* Scikit-learn
* Jupyter Notebook
